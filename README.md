# go todo app [勉強]

Go + Gin API, Vite + React + TypeScript + Tailwind UI

### 環境
- Go 1.21+
- Node.js 18+
- air (Makefile経由)

### エンドポイント
- App : `/ui/`
- API : `/api/v1/todos`

### 機能（予定）
- 優先度
- 〆切・期日
- calendar連携
- サブタスク
- 別タスクとの紐付け(?)

### めも

```cmd
// ビルド
make ui-build

// 開発
make dev

// air 直起動
make dev-fast
```
